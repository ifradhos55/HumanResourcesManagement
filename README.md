# Human Resources Management System

## Overview

The Human Resources Management System is a software application designed to manage employee records and payroll operations efficiently. It provides core HR functionality, including employee management, payroll calculation, and data export for external use.

This system is intended for educational and small-scale organizational use where straightforward HR and payment management is required.

## Features

* Add and manage employee records
* Update employee information
* Calculate payroll based on defined parameters
* Store employee and payroll data locally
* Export saved employee and payroll data to a CSV file for external analysis or record-keeping

## Functionality Breakdown

### Employee Management

* Add new employees to the system
* View and update existing employee information
* Maintain a structured employee dataset

### Payroll Calculation

* Calculate employee payments based on stored data
* Ensure consistent and repeatable payroll processing

### Data Export

* Export employee and payroll data into a CSV file
* Enables compatibility with spreadsheet software and external systems
